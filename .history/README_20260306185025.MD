# Checkout Process with Automated Framework (Python) 

## Prerequisites:
* .venv\Scripts\activate
* pytest --browser webkit --headed



Goal: A Python automation framework for a Shopify-based ordering experience that combines UI testing, API-based validation, and order verification for GamedayTLDR purchasing flows.

### Objects:

* Login/Logout
  * User is able to create an account, login to the app, and logout. *We will need to delete the users account at the end*
* CRUD (Create, Read, Update, Delete)
  * Login with an existing test account
  * Create a group, section, page, and then fill the page with a few phrases.
  * Validate the phrases exist.
  * Then delete the entire tree.
* API Endpoints
  * Ensure the test account has a functioning *free* API and an existing note.
  * Test a studio tool enhance a note.
  * Ensure enhance does not trigger an endpoint failure or failure to fetch.
